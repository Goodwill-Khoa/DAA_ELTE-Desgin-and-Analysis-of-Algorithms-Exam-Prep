# DAA ELTE – Design and Analysis of Algorithms (Exam Prep)

Interactive exam preparation web app for **ELTE MSc – Design and Analysis of Algorithms (DAA)**.

## About
This repository contains a small React + TypeScript single-page application built with **Vite**. It is intended to help with DAA exam preparation (mock exams, explanations, and learning materials).

## Tech Stack
- **Vite** (dev server & bundler)
- **React 18**
- **TypeScript**
- **React Router**

## Getting Started

### Prerequisites
- **Node.js** (recommended: latest LTS)
- **npm** (comes with Node)

### Install
```bash
git clone https://github.com/Goodwill-Khoa/DAA_ELTE-Desgin-and-Analysis-of-Algorithms-Exam-Prep.git
cd DAA_ELTE-Desgin-and-Analysis-of-Algorithms-Exam-Prep
npm install
